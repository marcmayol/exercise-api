Exercise API · v1 ● cargando…

API de ejercicios abierta, gratuita y multiidioma

Datos estáticos servidos por GitHub Pages (CDN, HTTPS, CORS abierto). Cada ejercicio incluye nombre en ES/EN, músculos principal/secundario, instrucciones, ilustración masculina y femenina y un mapa muscular SVG. Sin claves, sin límites.

Endpoints

GET  /v1/exercises.json              · índice de todos los ejercicios
GET  /v1/exercises/<slug>.json       · ficha completa (leg-press, plank, ...)
GET  /v1/muscles.json                · taxonomía de músculos (ES/EN)
GET  /v1/by-muscle/<muscle>.json      · ejercicios por músculo (chest, gluteal, ...)
     /images/<slug>-m.jpg | -f.jpg    · ilustración (hombre / mujer)
     /muscle-maps/<slug>.svg          · mapa muscular resaltado
Ejemplo: v1/exercises/leg-press.json
