Exercise API · v1 ● cargando…

API de ejercicios abierta, gratuita y multiidioma

Datos estáticos servidos por GitHub Pages (CDN, HTTPS, CORS abierto). Cada ejercicio incluye nombre en ES/EN, músculos principal/secundario, instrucciones, ilustración masculina y femenina y un mapa muscular SVG. Sin claves, sin límites.

Endpoints

GET /v1/exercises.json Índice de todos los ejercicios probar ↗
GET /v1/exercises/<slug>.json Ficha completa de un ejercicio leg-press ↗
GET /v1/muscles.json Taxonomía de músculos (ES/EN) probar ↗
GET /v1/by-muscle/<muscle>.json Ejercicios por músculo gluteal ↗
/images/<slug>-m.jpg | -f.jpg Ilustración (hombre / mujer) mujer ↗
/muscle-maps/<slug>.svg Mapa muscular resaltado SVG ↗
Ejemplo de uso
const base = "https://marcmayol.com/exercise-api";

// un ejercicio, en el idioma que quieras
const ex = await fetch(
  `${base}/v1/exercises/leg-press.json`
).then(r => r.json());

ex.name.es;                 // "Prensa de piernas"
ex.name.en;                 // "Leg press"
ex.primaryMuscles[0].es;    // "Cuádriceps"
ex.instructions.es;         // [ "Ajusta el asiento…", … ]
ex.images.female;           // .../leg-press-f.jpg
